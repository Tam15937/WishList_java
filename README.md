WishList - это веб сервис, который упрощает взаимодействие со списком желаемых подарков. Аналогичные сервисы предлагают больше возможностей только с сильным усложнением функционала, либо ограничениями в использовании.
Данный проект можно развернуть в своей сети и даже обеспечить доступ по интернету.
Из функционала:
1. Регистрация/Вход по никнейму и паролю с шифрованием пароля.
2. Просмотр имеющихся виш-листов
3. Просмотр списка подарков выбранного виш-листа
4. Обновление в реальном времени списков и подарков
5. Создание списка, тот кто создал помечается для списка как владелец.
6. Редактирование списка с сохранением состояния не изменившихся подарков (удаление, добавление, редактирование подарков, редактирования названия списка)
7. Удаление списка и входящих в него подарков.
8. Выход из аккаунта
9. Дальнейший функционал будет разрабатываться, и направление этого будет отражаться в TODO
